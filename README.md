# 学校食堂管理系统 python267

## 项目概述

学校食堂管理系统是基于 Python 语言和 Django 框架开发的一套适用于学校食堂的日常管理工作。该系统使用 SQLite 数据库，致力于提高食堂管理的效率。

## 技术栈

- **后端**: Python
- **框架**: Django
- **数据库**: SQLite
- **开发环境**: PyCharm

## 功能模块

- 食堂基本信息管理
- 菜品信息管理
- 订单管理
- 财务管理
- 用户权限管理

## 系统角色

- 系统管理员
- 食堂管理人员
- 普通用户

## 运行环境

- Python 3.x
- Django 3.x
- SQLite

## 部署步骤

1. 确保Python和Django环境安装正确。
2. 克隆代码到本地。
3. 导入数据库结构。
4. 运行 `python manage.py runserver` 启动开发服务器。
